# marquee-kit
better marquee animation for web.

这是一个简单的无缝滚动的功能。  
一般有这样的需求：一个抽奖功能，展示中奖用户列表，要不停的循环滚动中奖用户列表。  
那么这个就很合适了。  
在线 Demo ：[无缝滚动](http://dajbd.github.io/marquee-kit/)

采用 CSS3 Animation 实现。  
效果类似 html marquee tag 的滚动效果，但是比他的体验更好，因为这个是无缝滚动。
